# Good First Issues

AI-Agent-Cloud is designed to welcome new contributors with clear and approachable tasks.

## Beginner-friendly tasks
- Frontend polish: improve dashboard visuals, spacing, or component layout
- Documentation: update docs, landing page copy, or README sections
- New agents: add a new modular agent under `nexus-agents/`
- Observability: extend logging, metrics, or analytics flows
- Docker improvements: refine compose setup, environment handling, or local development
- Testing: add backend tests, frontend snapshot tests, or CI validation

## How to contribute
1. Choose an issue tagged `good first issue`
2. Fork the repository
3. Create a feature branch
4. Send a pull request with your improvements

## Who should contribute
- Developers learning open-source workflows
- AI engineers exploring modular systems
- Students building hands-on AI infrastructure experience
- Designers improving usability and docs
