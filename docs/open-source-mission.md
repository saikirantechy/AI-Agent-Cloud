# Open Source Mission

AI-Agent-Cloud is built around open-source values and community-first development.

## Why open source

- Community-driven innovation accelerates feature development.
- Transparency builds trust for a platform that coordinates AI workflows.
- Extensibility enables contributors to shape the ecosystem.
- Accessibility helps developers and teams adopt AI infrastructure without vendor lock-in.

## Community-driven innovation

- Encourage contributors to build new agents, integrations, and tools.
- Share architecture, roadmaps, and release plans with the community.
- Make collaboration central to the project vision.

## Developer ecosystem

- Provide onboarding docs, labels, and contribution pathways.
- Design the project to feel like a modern developer platform.
- Offer clear structure for plugin development and extension work.

## Transparency and extensibility

- Keep the platform open and modular.
- Document design decisions and future infrastructure plans.
- Align the project with open-source best practices.
