# Contributing to AI-Agent-Cloud

Thank you for your interest in contributing to AI-Agent-Cloud. This project is evolving into a professional open-source platform, and community contributions are essential to its success.

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Make your changes and run the project locally.
4. Submit a pull request with a clear description of your changes.

## Contribution Guidelines

- Keep changes small and focused.
- Write clear commit messages.
- Follow the existing code style in frontend and backend.
- Add documentation updates when introducing new features.

## Development Workflow

- Use `npm install` and `npm run build` in `frontend/`.
- Use `pip install -r backend/requirements.txt` in `backend/`.
- Use `docker compose up --build` for integration testing.

## Issue Reporting

- Open an issue for bugs, feature requests, or architectural discussions.
- Provide a clear description, reproduction steps, and expected behavior.

## Community Support

- Use GitHub issues for collaboration and feedback.
- Respect the open-source community and maintain a professional tone.
