# Security Policy

## Reporting a Vulnerability

If you discover a security issue in AI-Agent-Cloud, please report it through GitHub issues and mark it as a security concern.

## Secure Practices

- Do not commit secret keys or credentials to source control.
- Store secrets in environment variables or secure secret stores.
- Use `.env.example` as a template and keep `.env` private.
- Validate external input before passing it to AI or backend services.

## Responsible Disclosure

We appreciate responsible disclosure. Please include enough detail to reproduce the issue and avoid posting sensitive information publicly until it is fixed.
